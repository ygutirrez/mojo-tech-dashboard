## AI-Assisted Search Optimization
**Source:** [@edward.builds](https://www.instagram.com/reel/DTYpj_-jPB_/) | Jan 2026

Find and target the longer, conversational queries people are making with LLM help.

### Steps:
1. **Google Search Console** → Performance → Search Results
2. Add Filter → Query → **Regex**
3. Paste: `^(?:\S+\s+){6,}\S+$`
   - Finds searches with 7+ words (usually AI-assisted)
4. Pick a keyword you're already ranking for
5. Google it → grab **top 3 articles**
6. Use **Detailed Chrome extension** to copy all headings
7. Prompt ChatGPT:
   > "These are the headings for the top three rankings for [keyword]. Write headings for my own article that combines the best of everything."
8. Then: "Use this outline to write my SEO article for [keyword]"
9. **Before publishing:**
   - Add your own experience
   - Add logos of companies you've worked with
   - Add real photos/videos
   - Remove emdashes (ChatGPT tell)

### Other Useful Regex Filters:
**Informational searches:**
```
^(who|what|where|when|why|how|was|did|do|is|are|aren't|won't|does|if)[ ]
```

**E-commerce/Purchase intent:**
```
^(buy|order|purchase|get|best price|cheap|discount|deal|promo|offer|sale|where to buy|how much|cost of|price of)[ ]
```

### Tools Mentioned:
- Google Search Console
- Detailed Chrome Extension (for grabbing headings)
- ChatGPT
